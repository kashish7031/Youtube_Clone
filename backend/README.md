# YouTube Clone - MERN Stack Project

A full-stack video streaming application built using MongoDB, Express, React, and Node.js.

## Features
* **User Authentication:** JWT-based login and registration.
* **Video Management:** Upload videos (local files or URLs), delete videos, and view dashboard.
* **Video Player:** Hybrid player supporting MP4 uploads and YouTube embedded links.
* **Search & Filter:** Search videos by title.
* **Interactions:** Like, Dislike, and Subscribe functionality.

## Tech Stack
* **Frontend:** React, React Router, Axios
* **Backend:** Node.js, Express.js
* **Database:** MongoDB Atlas
* **File Storage:** Local server storage (Multer)

## Setup Instructions
1.  **Backend:**
    ```bash
    cd backend
    npm install
    npm start
    ```
2.  **Frontend:**
    ```bash
    cd frontend
    npm install
    npm run dev
    ```
